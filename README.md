# Neo Desk
Neo Desk is a AI-powered personal workspace.
You can work with AI's colleagues in the office software like Slack or Lark.

## Features
- Workspace
- Chat
- Task
- Calendar
- Document 
- AI Assistant

## Tech Stack

- Next.js
- TailwindCSS
- TypeScript
- Shadcn UI
- Better Auth
- Drizzle
- Neon
- React Query
- Zustand
